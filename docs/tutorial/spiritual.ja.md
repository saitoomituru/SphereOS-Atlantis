# スピリチュアル棚から入る

状態: `[ALPHA]`

この棚では、霊、神、祈り、象徴、依代、顕現を追放せず、その意味がどのWorldで、誰の定規により、
どの器へ接続されるかを記述します。Atlantis Coreは存在を否定も証明もせず、制定されたRegistryと
fact scopeを保持します。

## 言葉の橋

| この棚の言葉 | 開発上の接続候補 |
|---|---|
| 世界・界 | WorldとRegistry scope |
| 霊・神・使い魔 | World authorityが分類したAgency／Entity |
| 依代・札・祭具 | Meaningを保持または実行するVessel |
| 顕現・作用 | EventとOAE |
| 儀式・祈り | 入力、手順、制約、期待するEffect |
| 神学・宗派 | Causality ProfileまたはPresentation Profile |

この表は翻訳の強制ではありません。神を単なるプロセスへ、祈りを単なる関数へ縮退させず、
実装へ渡す際のBridge候補として使います。

## 最小演習

一つの作用について、次を別々に書きます。

```text
Meaning: その作用が世界内で持つ意味
World: どの世界・宗派・卓・作品の定規か
Agency: 誰または何が作用へ関与したと記録するか
Vessel: 共有・再現に使う媒体、手順、装置
OAE: 誰が、どの観測範囲で、何をEffectとして記録したか
unknown: 判断していないこと
```
