# 帰属表示ガイド

SphereOS Atlantisの文書、神話、UX、憲章を再利用するときは、対象ライセンスに従い、
媒体に合った合理的な形でOriginと変更を表示してください。

## CC BY 4.0対象の表示例

```text
「<対象作品名>」はSphereOS Atlantis contributorsによる作品を改変したものです。
Origin: https://github.com/saitoomituru/SphereOS-Atlantis
License: CC BY 4.0 https://creativecommons.org/licenses/by/4.0/
Changes: <翻訳、編集、追加、削除等>
```

## CC BY-SA 4.0対象の表示例

```text
「<派生憲章名>」はSphereOS Atlantis「意味と器の二重記述憲章」の翻案です。
Origin: https://github.com/saitoomituru/SphereOS-Atlantis
License: CC BY-SA 4.0 https://creativecommons.org/licenses/by-sa/4.0/
Changes: <翻訳、編集、追加、削除等>
```

帰属表示は、Originまたはcontributorsが派生物、世界観、製品、発言を推薦・公認したと
誤認させる形にしないでください。

コードの表示、再配布、NOTICE保持については、ルート[LICENSE](LICENSE)と[NOTICE](NOTICE)を参照してください。
